# Git Commit Convention
```bash
git commit -m "<type>(<scope>): <short summary>"
```

---

## Type
| Type       | Ý nghĩa                        |
|------------|--------------------------------|
| `feat`     | Tính năng / cập nhật mới       |
| `fix`      | Sửa lỗi                        |
| `docs`     | Cập nhật tài liệu              |
| `refactor` | Tái cấu trúc code (clean code) |
| `perf`     | Tối ưu                         |
| `test`     | Test                           |
| `chore`    | Tùy chọn khác                  |

 --- 

## Scope
**Ví dụ:** api, database, api, ...
